# Fix stairs
Fix stairs is a resource pack that adds 4 steps for all the stairs in minecraft. 

# Compatibility
The pack is compatible from 1.9 to the most recent version.

# FAQ
**Q: Do I need any sort of mod for this to work?**  
A: No, you don't. The resource pack is 100% minecraft vanilla.

**Q: Does it acctually have 4 steps?**  
A: No, its for decoration only.

**Q: The stair outline is shaped like a normal stair, is it normal?**  
A: Its not normal, but I can't do anything, there is no vanilla way to change the stair outline.

**Q: I have some lag with the resource pack enabled, is it normal**  
A: Yes, adding extra parts into a model, is making the game render more and that means  you will have a slight lose of fps.
